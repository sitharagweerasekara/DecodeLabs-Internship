# 🤖 Rule-Based AI Chatbot

A simple rule-based chatbot developed using Python as part of the **DecodeLabs Artificial Intelligence Internship – Project 1**.

The chatbot responds to predefined user inputs using conditional (`if-elif-else`) statements and runs continuously until the user chooses to exit. This project demonstrates the fundamentals of Artificial Intelligence through rule-based decision making.



## 📌 Project Overview

This chatbot simulates a basic conversation by recognizing predefined keywords and returning appropriate responses. It does not use Machine Learning or Natural Language Processing, making it an excellent introduction to AI logic and control flow.



## ✨ Features

- 👋 Responds to greetings
- 💬 Answers predefined questions
- 🧠 Uses rule-based decision making
- 🔄 Runs continuously until the user exits
- 🚪 Handles exit commands gracefully
- ⚠️ Responds to unknown inputs with a default message



## 🛠️ Technologies Used

- Python 3
- Conditional Statements (`if`, `elif`, `else`)
- Loops (`while`)
- String Manipulation
- Basic Input/Output
