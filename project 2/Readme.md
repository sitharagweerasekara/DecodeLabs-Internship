# 🌸 AI Project 2 - Iris Flower Classification

A machine learning classification project developed as part of the DecodeLabs Artificial Intelligence Internship.

## Features

- Load the Iris dataset
- Split data into training and testing sets
- Standardize features
- Train a K-Nearest Neighbors classifier
- Evaluate model performance
- Generate confusion matrix and classification report

## Technologies

- Python
- Scikit-learn
- Machine Learning
- KNN Classifier

## Dataset

The project uses the built-in Iris dataset from scikit-learn.

## Results

- Accuracy
- Confusion Matrix
- Classification Report
