# AI Recommendation System

## Overview

This project is developed as part of the DecodeLabs AI Internship (Project 3).

The system recommends courses based on the user's interests using Natural Language Processing techniques.



## Features

- User preference input
- TF-IDF vectorization
- Cosine Similarity
- Course recommendation
- Match confidence score



## Technologies Used

- Python
- Pandas
- Scikit-learn
